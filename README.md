# 🍽️ 特朗普吃屎大作战

一个简单有趣的 HTML5 接物小游戏。

## 在线试玩

直接用浏览器打开 `index.html` 即可开始游戏。

## 游戏玩法

- 使用 **← → 方向键** 或 **A D 键** 移动盘子
- 移动端点击屏幕左右按钮控制
- 接住从天而降的便便获得分数
- 便便速度会随着分数增加而加快

## 便便类型

| 类型 | 外观 | 分数 | 速度 | 出现概率 |
|------|------|------|------|----------|
| 普通便便 | 棕色 | 10分 | 慢 | 60% |
| 高级便便 | 红褐色 | 30分 | 中 | 30% |
| 稀有便便 | 金色 | 50分 | 快 | 10% |

## 升级机制

| 条件 | 奖励 |
|------|------|
| 累计 500 分 | 盘子加倍（2个盘子） |
| 累计 10000 分 | 三盘齐发（3个盘子） |

## 游戏特性

- 生命值系统：初始5条命，失误扣1命，接住10个便便奖励1命
- 多盘子升级：500分双盘，10000分三盘
- 速度递增：每生成5个便便速度增加1%
- 中英文语言切换
- 自定义便便图片

图片使用 Base64 存储在浏览器本地存储中，刷新页面不会丢失。
