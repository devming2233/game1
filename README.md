# 🎁 接礼盒大作战

一个简单有趣的 HTML5 接物小游戏。

## 在线试玩

直接用浏览器打开 `index.html` 即可开始游戏。

## 游戏玩法

- 使用 **← → 方向键** 或 **A D 键** 移动盘子
- 移动端点击屏幕左右按钮控制
- 接住从天而降的礼盒获得分数
- 礼盒速度会随着分数增加而加快

## 礼盒类型

| 类型 | 外观 | 分数 | 速度 | 出现概率 |
|------|------|------|------|----------|
| 普通礼盒 | 红色 | 10分 | 慢 | 60% |
| 高级礼盒 | 青色 | 30分 | 中 | 30% |
| 稀有礼盒 | 金色 | 50分 | 快 | 10% |

## 升级机制

| 条件 | 奖励 |
|------|------|
| 累计 500 分 | 盘子加倍（2个盘子） |
| 累计 10000 分 | 三盘齐发（3个盘子） |

## 自定义功能

点击 **"自定义图片"** 按钮可以：
- 更换盘子图片
- 更换三种礼盒的图片

图片使用 Base64 存储在浏览器本地存储中，刷新页面不会丢失。
